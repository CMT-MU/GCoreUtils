Eigenvalue problem#

solve eigenvalue problem for a set of parameters.

eigensystem(mat_func, params, eigenvector=True)[source]#

solve eigenvalue problem.

Parameters:

  • mat_func (function) -- a function to get a matrix for a given set of parameters

  • params (np.array) -- a set of parameters in each row

  • eigenvector (bool, optional) -- compute eigenvectors ?

Returns:

np.array: eigenvalues of a set of parameters in each column np.array, optional: eigenvector matrices in order of a set of parameters, eigenvector of a matrix in each column

Return type:

tuple
