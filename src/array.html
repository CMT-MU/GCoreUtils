Notes: string / internal

  • scalar : s, {s}, {{s}}, ... / s, [s], [[s]], ...

  • vector : v = [s], {v} / [s], [[s]].

  • matrix : m = [[s]], {m} / [[s]], [[[s]]].

classmethod grid(grid_min, grid_max, grid_n, endpoint=False)[source]#

create grid points.

Parameters:

  • grid_min (list) -- lower bounds in each direction.

  • grid_max (list) -- upper bounds in each direction.

  • grid_n (list) -- number of points in each direction.

  • endpoint (bool, optional) -- include end point ?

Returns:

grid points.

Return type:

NSArray

Notes

  • grid point: increase of indices from left to right.

  • x[i] = (max-min)*i/n for endpoint is False, otherwise (max-min)*i/(n-1).

__init__(data=None)[source]#

initialize the class.

Parameters:

data (dict, optional) -- { name: ( [column], [role], [[value]] ) }

Notes

  • value can be omitted.

  • if [role] is None, None is used, while if [role] is str, str is used for all columns.

  • if [[value]] is None, empty data frame is created.
